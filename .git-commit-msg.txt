feat: 힌트 시스템 개선 및 차트 가시성 향상

## 주요 변경사항

### 힌트 시스템 개선
- 중복된 "기본 가이드" 힌트 제거 (가이드라인 섹션과 중복)
- 3단계 힌트를 2단계로 재구성:
  - 1단계: 문제 분석 (기존 2단계 내용)
  - 2단계: 정답 코드 전체 제공 (학습 목적)
- 힌트 내용 개선:
  - Case 1: 이미지에 보이는 텍스트만 전달하도록 안내
  - Case 2: 장식용 이미지 처리 방법 안내
  - Case 3: 차트의 연도와 금액 정보 모두 포함 안내

### 차트 SVG 가시성 개선
- SVG 높이 400px → 450px로 증가 (연도 표시 공간 확보)
- 연도 텍스트 크기 16px → 18px로 증가
- 연도 텍스트 위치 조정 (y: 350 → 330)로 막대 아래 명확히 배치
- 막대 위치 조정하여 연도 텍스트와 겹치지 않도록 개선

## 개선 효과
- 힌트 시스템이 더 간결하고 실용적으로 개선됨
- 차트의 연도 정보가 명확하게 보이도록 개선됨
- 학습 목적에 맞게 정답 코드 전체를 제공하여 학습 효과 향상
